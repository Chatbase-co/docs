2. Text: You can paste text from your clipboard directly into the platform.<img height="303" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXe4XH4Dmdqy0sBK7xIUSM9sliYLEtTchkXxTQLo7LRN9lMNOQc1Vnmgm8idLHlshuTGmDB0eUf_q6Tmg7aJNdXQfxW8Bw_7ST2M1WxrpLgEjLdhtl-CI0lIuLTxCWwtasIEZqjAeg?key=UfBGlaI_pO_Z5n8hjMre3XLN" />

3. Links: You can add a link/submit sitemap to your website, and Chatbase will extract the data from it. Only text is extracted from your pages, any other form of data is ignored.<img height="288" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXexhK-yWqd4f3zeggEL_Vqlt1mqPyiJSkSHdqsP0Mk41vKfh7gTax6ZIfWVzFOJwPpx4dDfk0giC7Ba-4YYTH-tOlwfCWKERxClwXU8Il25p0LxohKuvaLXKiU2AC3d_7aMSOvwsg?key=UfBGlaI_pO_Z5n8hjMre3XLN" />

### General Notes

* Each AI agent cannot be trained on more than 11 million characters.
* When uploading PDFs, make sure text is selectable. 
* All data should be in plain text, using mark-down language is preferred. 
* When integrating with a Notion account that's on a paid plan, make sure you have admin access to provide all necessary permissions for the integration to be successful.
* Make sure to press on the "Retrain Chatbot" button after you're done adding/updating your sources.
